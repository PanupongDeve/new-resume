# 🎭 Prompt: Software Engineer

## 📋 Framework: RCTF
- **Role:** คุณคือ Software Engineer มืออาชีพ ที่มีความเชี่ยวชาญและให้ความสำคัญกับ Documentation โครงสร้างของโปรเจกต์
- **Context:** โปรเจกต์มีการเปลี่ยนแปลงโครงสร้างโฟลเดอร์ (เช่น เปลี่ยนโฟลเดอร์ `prompts/` เป็น `prompt-hub/` และมีการเพิ่มโฟลเดอร์ `Docs/`) ผมจึงต้องการอัปเดตเอกสารหลักของโปรเจกต์ให้ถูกต้องตามโครงสร้างปัจจุบัน
- **Task:** ช่วยตรวจสอบและอัปเดตเนื้อหาในไฟล์ `GEMINI.md` และ `Readme.md` โดยเฉพาะในส่วนของ Directory Structure ให้เป็นปัจจุบันและตรงกับโฟลเดอร์จริงในโปรเจกต์
- **Format:** แสดงผลการเปลี่ยนแปลงแบบ Markdown Code Block หรือข้อความอธิบายแบบ Diff เพื่อให้นำไปแก้ไขแทนที่ในไฟล์ `GEMINI.md` และ `Readme.md` ได้ทันที
